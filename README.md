# PHP 與 MySQL 購物車系統

使用 **PHP** 與 **MySQL** 所製作的雙角色電商購物平台。系統實作了從前端商品瀏覽、購物車動態增刪，到後端訂單成立的完整商務邏輯，並透過權限控管精準區分「使用者」與「管理者」雙介面。

## 🛠️ 核心功能

* **會員身份驗證與權限控管**：利用 Session 機制實作會員註冊、登入系統，並精準隔離前端消費者與後台管理者的操作權限。
* **動態購物車邏輯 (CRUD)**：使用者可自由瀏覽商品，並於跨網頁間即時對購物車進行新增、修改數量與刪除商品。
* **結帳流程與關聯式訂單建立**：實作流暢的結帳防錯流程，並在 MySQL 資料庫中自動建立多表關聯的歷史訂單紀錄。
* **後台商務管理系統**：設計獨立的管理員後台，提供商品上架與訂單狀態變更功能，具備初級 ERP 系統雛形。
